# Aplikasi Konversi Suhu Python

Aplikasi sederhana berbasis Python untuk melakukan konversi suhu dari satu satuan ke satuan lainnya seperti Celsius, Fahrenheit, Reamur, dan Kelvin.

## Fitur

* Konversi Celsius ke Fahrenheit
* Konversi Fahrenheit ke Celsius
* Konversi Celsius ke Kelvin
* Konversi Kelvin ke Reamur
* Program berbasis terminal (CLI)

## Teknologi

* Python

## Cara Menjalankan Program

1. Pastikan Python sudah terinstall di komputer.
2. Jalankan file program dengan perintah:

```
python konversi_suhu.py
```

## Contoh Output

Program akan meminta pengguna memasukkan nilai suhu dan memilih jenis konversi yang diinginkan, lalu menampilkan hasil konversinya.
